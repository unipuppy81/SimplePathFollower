# 🗺 Simple Path Follower (Motion Sketcher)

**Simple Path Follower**는 유니티에서 베지어(Bézier) 곡선 기반의 이동 경로를 시각적으로 설계하고 관리하는 강력한 도구입니다. 데이터 중심 설계(`ScriptableObject`)를 통해 경로를 자산화하고, 다양한 오브젝트에 유연하게 적용할 수 있습니다.

---

## 💠 1. Introduction: 베지어 곡선(Bézier Curve)의 원리

유니티의 일반적인 애니메이션 방식은 키프레임 사이의 보간이 최단 거리로만 이루어져 일정한 속도를 유지하거나 정교한 곡선을 설계하기 어렵습니다. 본 프로젝트는 이 문제를 해결하기 위해 **베지어 곡선**을 활용합니다.

### 📍 곡선의 기하학적 구현
베지어 곡선은 직선 위를 이동하는 점들의 비율적 위치를 연결하여 생성됩니다.

1. **점의 이동 (t):** 하나의 직선 위를 점 $M$이 일정 속도로 이동한다고 가정합니다. 이때 $t$는 선분 위를 비율적으로 얼마나 나아갔는지를 나타내는 수치($0 \le t \le 1$)입니다.
2. **선의 확장:** 직선을 하나 더 추가하고 그 위에 동일한 규칙으로 움직이는 점 $M_0$와 $M_1$을 놓습니다.
3. **궤적의 생성:** 이동하는 두 점 $M_0$와 $M_1$을 잇는 새로운 선을 긋고, 그 선 위를 다시 일정 속도로 이동하는 점 $B$를 놓습니다. 
4. **결과:** 점 $B$가 그리는 궤적을 살펴보면 부드러운 곡선이 나타납니다. 이를 **2차 베지어 곡선(Quadratic Bezier Curve)**이라고 합니다.

조절점(Control Point)을 늘릴수록 3차, 4차 등 더 정교한 곡선을 만들 수 있으며, 본 프로젝트는 더욱 유연한 곡률 제어를 위해 3차 베지어 곡선을 기반으로 설계되었습니다.


### 🔢 수학적 공식 (Cubic Bézier Formula)
3차 베지어 곡선은 네 개의 점($P_0, P_1, P_2, P_3$)을 이용해 다음과 같은 다항식으로 표현됩니다.

$$B(t) = (1-t)^3P_0 + 3(1-t)^2tP_1 + 3(1-t)t^2P_2 + t^3P_3, \quad 0 \le t \le 1$$

이 공식은 매끄러운 곡선을 보장할 뿐만 아니라, 프로그래밍적으로 경로를 세밀하게 샘플링하여 일정한 이동 속도를 구현하는 기초가 됩니다.

## 💠 2. 특징 (Features)

* **Visual Editing (Scene View):** 에디터에서 핸들을 직접 조작하여 경로를 설계합니다. `Shift + Left Click`으로 지면에 즉시 노드를 추가할 수 있어 작업 효율이 높습니다.
* **Data-Driven (ScriptableObject):** 경로 데이터를 `.asset` 파일로 관리하여 메모리 오버헤드를 줄이고, 여러 오브젝트가 동일한 경로를 공유하거나 재사용할 수 있습니다.
* **Flexible Speed Control:** 각 노드(Node)마다 독립적인 속도 값을 부여할 수 있어, 직선 구간은 빠르게 곡선 구간은 천천히 움직이는 등 정교한 완급 조절이 가능합니다.
* **Intelligent Rotation (LookAt):** `lookAtTarget`이 지정되면 해당 타겟을 응시하며 이동하고, 없을 경우 진행 방향을 자동으로 계산하여 자연스럽게 바라봅니다.
* **Seamless Loop:** `isLoop` 옵션을 통해 경로의 끝점에서 시작점으로 부드럽게 돌아가는 무한 반복 기능을 지원합니다.
* **Decoupled Event System:** 각 노드에 이벤트를 부여하여 특정 지점 도달 시 사운드, 이펙트, 퀘스트 로직 등을 실행하는 브릿지 역할을 수행합니다.

## 💠 3. Component Guide

### 🔘 Path Creator
경로의 시각화와 데이터 세이브/로드를 담당합니다.
- **Batch Tools:** 경로 전체를 특정 위치로 이동(Offset)시키거나 속도를 일괄 수정하는 유틸리티를 제공합니다.

### 🔘 Path Follower
오브젝트의 실제 물리적 이동을 처리합니다.
- **Rotation Logic:** `Slerp`를 이용한 부드러운 회전과 안정적인 방향 제어를 제공합니다.

### 🔘 Path Manager (Example)
다양한 경로 자산을 중앙에서 제어하는 예시 클래스입니다.
- **PlayPath(index):** 리스트에 등록된 여러 경로 중 원하는 경로를 선택하여 실시간으로 이동을 시작합니다.

## 💠 4. 활용 예시 (Use Cases)
* **컷신 카메라 (Cutscene):** 인물의 동선에 따라 부드럽게 움직이며 특정 타겟을 응시하는 카메라 연출
* **환경 오브젝트:** 정해진 궤적을 순찰하는 몬스터나 일정한 패턴으로 움직이는 이동 발판
* **이벤트 연출:** 특정 지점 도달 시 이벤트를 발생시켜 연출용 이펙트나 사운드 재생
